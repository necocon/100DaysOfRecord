# 100DaysOfRecord
勉強日数を記録してペーストできるChrome拡張機能です。

<hr>
## Usage
Chromeウェブストア(URL)
1. 拡張機能アイコンをクリック
2. 日数をカウント
3. 入力エリアで右クリックメニューを出して「日数とハッシュタグを貼り付け」を選択

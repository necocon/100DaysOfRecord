<h1>100DaysOfRecord</h1>
勉强日数をカウントして、日数と100DaysOfCodeのハッシュタグをクリップボードにコピーできる拡張機能です。

<h2>Usage</h2>

<ol>
<li>拡張機能アイコンをクリック</li>
<li>日数をカウント</li>
<li>COPYボタンを押すと日数とハッシュタグがクリップボードにコピーされる</li>
</ol>
